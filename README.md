# Theatrical Players Refactoring Kata

This project is an implementation of the Theatrical Players Refactoring Kata, designed to practice and demonstrate refactoring and clean code techniques in C#.

## Overview

The application generates a statement for a customer based on a set of theatrical play performances. It calculates the total amount owed and volume credits earned, supporting different play types (Tragedy, Comedy) with their own pricing and credit rules.
